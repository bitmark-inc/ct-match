# Pfizer projects

Contains 

## Pfizer simulator
### Prerequisite

- Go 1.9+

### How to build

Put the project in `$GOPATH/src/github.com/bitmark-inc/pfizer`
```
go build
```

### Configuration

Please use the testnet config file `/flow-simulator/testnet.conf` for your settings.

### Run

```
$ flow-simulator -c testnet.conf
```
